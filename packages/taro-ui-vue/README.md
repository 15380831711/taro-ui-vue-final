## 项目背景 
`tarojs` 已经开始支持 `vue` 去写 taro 应用了，由于缺少相关的 ui 库，因此决定 按照 taro-ui 重写成 `vue` 组件。
形成了 `taro-ui-vue` 这个库。

## 使用注意
Taro Ui Vue 提供的是源文件，源文件部分代码使用 `ts` 编写，需要项目支持 ts。
如果是用 `taro init` 命令创建，则需在使用时选择 `ts` 。
